 # Types of testing (developer)
   - Unit Testing - Test a React Component in Isolation  (Specific Component) - Testing 1 Unit/Component in Isolation
   - Integration Testing - Testing the Integration of the Component
   - End to End Testing - e2e testing - Testing a React App As soon as user land to the website to user leave the website. (Tools required for e2e testing - Selenium, Cypress, Puppeteer)

 # Setting up Testing in our app
   - Installed React Testing Library
   - Installed Jest
   - Installed Babel dependencies
   - Configure Babel
   - Configure Parcel Config file to disable default babel transpilation
   - Jest Configuration - npx jest --init
   - install jsdom library - (If you're using Jest 28 or later, jest-environment-jsdom package now must be installed separately.)
   - Install npm i -D @babel/preset-react - to make JSX work in test cases.
   - Include @babel/preset-react inside my babel config
   - Install npm i -D @testing-library/jest-dom
